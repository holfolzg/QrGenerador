# ⚡ QR Generator Pro - React + Vite

Una aplicación web moderna, rápida y minimalista para generar códigos QR de forma instantánea. Construida con **React** y potenciada por **Vite** para una experiencia de desarrollo y usuario de alto rendimiento.

---

## ✨ Características

* 📥 **Descarga Directa:** Exporta tus códigos en formato PNG de alta calidad.
* 📱 **Totalmente Responsive:** Diseño adaptado para móviles, tablets y escritorio.
* 🧹 **Interfaz Limpia:** Sin anuncios, enfocada totalmente en la utilidad.

## 🛠️ Stack Tecnológico

* [React.js](https://reactjs.org/) - Biblioteca de UI.
* [Vite](https://vitejs.dev/) - Entorno de desarrollo ultrarrápido.
* [qrcode.react](https://github.com/zpao/qrcode.react) - Componente React para la renderización de QR.

## 🚀 Instalación y Uso

Sigue estos pasos para ejecutar el proyecto localmente:

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/holfolzg/GeneradorQR.git](https://github.com/holfolzg/GeneradorQR.git)
    ```

2.  **Entra al directorio:**
    ```bash
    cd generateQr
    ```

3.  **Instala las dependencias:**
    ```bash
    npm install
    # o si usas yarn
    yarn install
    ```

4.  **Inicia el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

5.  **Abre tu navegador en:** `http://localhost:5173`
